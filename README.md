# FinShield

FinShield is a Banking Management System developed using Flask and SQLite.

## Features

- User Registration
- Login Authentication
- Account Dashboard
- Savings Information
- Transaction History
- Fraud Detection Alerts
- User Profile Management

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## Run Locally

pip install flask

python init_db.py

python app.py

Open:
http://127.0.0.1:5000
